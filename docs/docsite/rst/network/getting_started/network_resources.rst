GitHub repos
============

Assible hosts module code, examples, demonstrations, and other content on GitHub. Anyone with a GitHub account is able to create Pull Requests (PRs) or issues on these repos:

- `Network-Automation <https://github.com/network-automation>`_ is an open community for all things network automation. Have an idea, some playbooks, or roles to share? Email assible-network@redhat.com and we will add you as a contributor to the repository.

- `Assible collections <https://github.com/assible-collections>`_ is the main repository for Assible-maintained and community collections, including collections for network devices.
