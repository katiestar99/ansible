**********************************
Network Advanced Topics
**********************************

Once you have mastered the basics of network automation with Assible, as presented in :ref:`network_getting_started`, use this guide understand platform-specific details, optimization, and troubleshooting tips for Assible for network automation.

**Who should use this guide?**

This guide is intended for network engineers using Assible for automation. It covers advanced topics. If you understand networks and Assible, this guide is for you. You may read through the entire guide if you choose, or use the links below to find the specific information you need.

If you're new to Assible, or new to using Assible for network automation, start with the :ref:`network_getting_started`.
